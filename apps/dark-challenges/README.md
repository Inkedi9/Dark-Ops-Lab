# DarkChallenges

DarkChallenges is an unguided offensive security training platform focused on hands-on exploitation.

No walkthroughs. No hints by default. Only signals, feedback and progression.

## Features

- Injection challenges (SQLi, XSS, Command Injection)
- Visual sandbox simulation (browser, terminal, database)
- Real-time terminal feedback system
- XP, levels and rank progression
- Challenge packs and structured learning paths

### CTF Mode

- Multi-step scenarios
- Flag fragments & final flag validation
- Dedicated badges and XP rewards

### Warzone Mode

- Live attack simulations
- Dynamic logs and hostile signals
- Detection risk system
- Timer-based pressure
- Failure state (lockdown)

## Tech

- Next.js (App Router)
- React
- TailwindCSS
- Local storage persistence (no backend required)

## Philosophy

DarkChallenges is designed to simulate offensive thinking:

- Explore signals
- Test hypotheses
- Break systems
- Learn by failure
